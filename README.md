# FUTURE_DS_01

## 📊 Business Sales Performance Analytics

## 📌 Project Overview
This project analyzes business sales data to identify trends, top-performing products, and regional performance.

## 🛠 Tools Used
- Tableau

## 🔍 Analysis Performed
- Sales trend over time
- Category-wise performance
- Region-wise sales
- Top 10 products

## 📈 Key Insights
- Sales are increasing over time  
- Technology category performs best  
- West region generates highest sales  

## 💡 Recommendations
- Focus on high-performing categories  
- Expand in top regions  
- Promote top-selling products
